 # Windows 11 Home
 ## Add license key:
 ```
 slmgr /ipk TX9XD-98N7V-6WMQ6-BX7FG-H8Q99
 ```
 ## Add KMS server:
 ```
 slmgr /skms kms8.msguides.com
 ```
 or
 ```
 slmgr /skms kms9.msguides.com
 ```
 or
 ```
 slmgr /skms s8.uk.to
 ```
 or
 ```
 slmgr /skms s9.us.to
 ```
 ## Activate:
 ```
 slmgr /ato
 ```
